# Game of Thrones

## Usage
To run the application locally, navigate to the app directory using the command line and run the commands `npm i` and then `npm start`.

## Technologies
This project uses 
* React, 
* JavaScript, 
* CSS

and uses data from [An API of Ice And Fire](https://anapioficeandfire.com/)
